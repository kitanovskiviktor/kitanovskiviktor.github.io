Overview

JavaScript is one of the most popular programming languages and it is growing faster than any other programming languages. In this training you will learn the fundamental concepts of JavaScript as a key tool for front-end, back-end and full-stack developers. This course will cover concepts or topics as variables, loops, conditions, functions and function constructors (ES6 features) and much more.

Learn the JavaScript language, but also learn how to program, how to solve problems. This course will take you to the next level and learn the React framework based on this language which is the most required these days.

What you Will Learn?

Fundamental concepts in JavaScript

Learn JavaScript functions, Javascript loops, Javascript variables, Javascript condition structure

Understanding "let" and "const";

CSS basics

Modern JavaScript: ES6+

Understanding Classes, Properties and Methods
